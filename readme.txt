Приложение KeyCodes

1. Создать разметку в index.html 
2. Добавить стили в style.css
3. Подключите внешний шрифт
4. Приложение должно быть размещено по центру экрана - компактно
5. Написать логику в script.js
6. Стартовый экран приложения должен запрашивать пользователя, что бы он нажал любую клавишу на клавиатре, дальше на экране показывается нажатая клавиша (ключ и код) и так далее нажимая любую клавишу.
